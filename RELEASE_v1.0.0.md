# BioViz Local v1.0.0 发布说明

我们非常高兴地宣布 **BioViz Local v1.0.0** 正式发布！这是首个稳定版本，标志着本地化生物通路分析与 AI 辅助研究的深度融合。

## 🎉 核心亮点

### 1. **Logic Lock 安全架构** 🛡️
引入了首创的 AI 安全协议，确保 AI 智能体在执行关键操作时受到严格的逻辑约束。
- **绿区**：只读操作（如查询通路列表）自动执行。
- **黄区**：数据处理操作（如分析、导出）需用户确认。
- **红区**：禁止操作，完全阻断潜在风险。

### 2. **上下文感知 AI 助手** 🧠
AI 不再是简单的聊天机器人，而是您的科研搭档。
- **全数据洞察**：AI 现在可以读取通路中所有 30+ 个基因的完整表达数据进行分析。
- **自动工具调用**：AI 可自主执行通路渲染 (`render_pathway`)、统计分析 (`get_pathway_stats`) 等操作。
- **智能追问**：执行操作后，AI 会自动基于新结果提供进一步的生物学解释（如切换到凋亡通路后自动分析）。

### 3. **交互式通路可视化** 🔬
- **动态渲染**：支持 KEGG 通路的高保真渲染，实时映射 Log2FC 表达值。
- **多通路漫游**：支持在不同通路间无缝切换（AI 辅助或手动选择）。
- **点击交互**：点击任意节点可查看详细的基因信息和表达数值。

## 📋 完整变更日志

### 新增功能
- **AI Chat 面板**：基于上下文的智能对话界面，支持 Markdown 渲染和工具调用反馈。
- **Insight Badges**：在分析结果旁自动生成 AI 关键发现徽章（如 "🔥 High Activity"）。
- **数据导入向导**：智能识别 CSV/Excel 文件中的 Gene Symbol, Log2FC 和 P-value 列。
- **火山图交互**：与通路图联动的交互式火山图，支持框选和悬停查看。

### 优化与修复
- **性能优化**：主要分析逻辑迁移至本地 Python 引擎，大幅提升处理速度。
- **DeepSeek 集成**：原生支持 DeepSeek-V3 等高性能模型，降低使用成本。
- **热重载修复**：修复了 sidecar 连接断开的问题，确保长时间运行的稳定性。
- **文档完善**：提供了完整的中文 README 和配置指南。

## 🚀 升级指南

如果您是从测试版升级，请执行以下操作：
1. 拉取最新代码：`git pull`
2. 更新依赖：`npm install`
3. 重新配置 AI：检查 `.env` 文件配置是否符合新格式。
4. 重新构建：`npm run tauri build`
